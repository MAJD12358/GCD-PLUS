# Contributing to GCD+

Thank you for considering contributing to GCD+! Before you start, please read the following guidelines.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature/bugfix: `git checkout -b feature-name`.
3. Make your changes and commit them: `git commit -m 'Add new feature'`.
4. Push your changes to your fork: `git push origin feature-name`.
5. Create a pull request (PR) to the `main` branch of the main repository.

## Code Style
Please follow the established code style for GCD+ to maintain consistency.

## Report Issues
If you find any issues or have suggestions, please open an [issue](https://github.com/yourusername/gcdplus/issues).

Thank you for your contributions!
